基本环境

Win 10 v1709

VSCode 安装 ESLint 扩展

打开 VSCode 搜索 ESLint 扩展安装

NPM 安装 ESLint

  1. 全局安装（推荐）:

    npm install eslint -g

  2. 项目根目录本地安装：

    npm install eslint --save-dev

1. 自定义配置规则

  1. 全局配置
    在当前用户根目录下执行 eslint --init,根据个人爱好选择生成配置文件 .eslintrc.json
    例如：windows\user\username

  2. 项目根目录配置：
