## 完整 HTML 树

![DOM](https://i.loli.net/2017/10/26/59f1ff406e5f5.png)
* 相关的 CSS 文件（针对样式）：

```html
<link rel="stylesheet" type="text/css" href="style.css">
```

* 相关的 JavaScript 文件（更改渲染和行为的多用途脚本）：

```html
<script src="myjavascripts.js"></script>
```

* 粗体斜体

```html
<!-- 推荐使用<strong>进行加粗  -->
 <strong>粗体</strong>
<!-- 推荐使用<em>进行斜体显示  -->
<em>斜体</em>
```

* `<sup>` 标签定义上标文本。
* `<sub>` 标签定义下标文本。
